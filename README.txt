# Mahsa Hostel Portal

This is a web-based hostel management system for MAHSA University students and admins.

## How to Run
1. Install XAMPP and start Apache + MySQL
2. Place the project folder in: /Applications/XAMPP/xamppfiles/htdocs/
3. Open browser and go to: http://localhost/mahsa_website/login.html
4. Create database `mahsa_hostel_db` in phpMyAdmin
5. Import tables: students, admins, passes, feedback, maintenance

## Features
- Secure login for students and admins
- Forgot password with email reset
- Admin dashboard to manage student info and reply to requests
- Student dashboard to view info, submit issues, and create hostel pass
- Hostel pass system with color status and admin approval
- Responsive design for desktop and mobile
